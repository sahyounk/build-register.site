# Build-Register website starter

This package gives you a clean static site structure for GitHub + Cloudflare Pages.

## Structure

- `index.html` — landing page
- `app/index.html` — product/demo preview
- `assets/logo-build-register.svg` — reusable logo asset

## Recommended repo setup

1. Create a GitHub repo named `build-register-site`.
2. Upload these files to the repo root.
3. In Cloudflare Pages, switch from direct upload to **Connect to Git**.
4. Select the repo and deploy.
5. Keep `build-register.com` and `www.build-register.com` attached as custom domains.
